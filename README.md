## How to install app?

To install app run following commands:

- git clone https://github.com/sanzharanarbay/museum.git.
- cp .env.example  .env.
- Open .env and configure DB settings.
- composer install.
- php artisan key:generate.
- php artisan migrate.
- php artisan db:seed.
- php artisan optimize(optionally).
- php artisan serve.
